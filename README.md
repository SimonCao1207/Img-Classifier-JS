# Img-Classifier-JS

## Goal

- Build an image classification with `tensorflowJS`

## Run demo
- `cd` into `local-server`
        
        npm install
        npm start 

- Serving at `http://localhost:8081/predict.html` (port number can be changed in `server.js`)
